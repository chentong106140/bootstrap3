我在中间 col-md-pull-2: pull向左拉2个，呈现在最左边（如何确定是2呢？，由于当前是4，它要与它左边的列调换位置，就直接与左边位置的列交换数值=2），定位在左边
我在右边 col-md-pull-2: pull向左2个，呈现在中间（如何确定是2呢？原本的中间那一列是4跑到最左边了，而当前列是6，所以12-4-6=2，正好满足一行）
