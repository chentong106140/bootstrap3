4-表单控件

       
        最常见的表单文本字段是输入框 input。用户可以在其中输入大多数必要的表单数据。
        Bootstrap 提供了对所有原生的 HTML5 的 input 类型的支持，
        包括：text、password、datetime、datetime-local、date、month、time、
        week、number、email、url、search、tel 和 color。
        适当的 type 声明是必需的，这样才能让 input 获得完整的样式。
